WayFar travel web application gives travel services to its customers to explore and experience different parts of the world. It gives details about popular destinations and attractive offers on different tour packages by giving rich experience to its customers.

WayFar app provides below services for its customers.

Popular Destinations

Tour Packages

Offers on Packages

Gallery 

The source code of the application can be downloaded through this link.WayFar.zip

The entry point of the application is login.html. Open the login.html using live server of Visual Studio Code IDE.

You can login to the application with default values of both username and password as "admin".
